# P2: Creative workflow MVP with lineage, compare, templates, and experiments

## Goal

Build differentiated creative workflow features on top of existing jobs, gallery, and Prompt Square.

## Requirements

- Add image/job lineage for variant generation and detail display.
- Add multi-profile/model compare generation and side-by-side results.
- Add prompt template variables and deterministic expansion.
- Add favorite/tags/experiment tracking to gallery items.

## Acceptance Criteria

- [ ] A generated image can be used as a parent and lineage can be traced.
- [ ] Compare API does not break existing queue/quota behavior.
- [ ] Template expansion is deterministic and tested.
- [ ] Gallery can filter favorites/tags/experiments.

## Out of Scope

- Realtime collaboration.
- Community comments/sharing.
- Workflow engine introduction.
